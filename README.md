# PyTutor

PyTutor es una aplicación web para gestionar tutorías particulares, estudiantes, clases, pagos y rentabilidad.

## Objetivo

El objetivo del proyecto es ayudar a organizar clases particulares, registrar estudiantes, calcular cobros automáticamente según reglas de negocio, guardar logs de clase y analizar ingresos por estudiante.

## Funcionalidades iniciales

- Registro de estudiantes.
- Registro de clases individuales y grupales.
- Cálculo automático del valor de la clase.
- Descuentos por primera clase.
- Descuentos por grupo.
- Registro de pagos.
- Historial por estudiante.
- Log de lo trabajado en cada clase.
- Reportes de ingresos.

## Tecnologías planeadas

- Backend: Spring Boot
- Frontend: Angular
- Base de datos: PostgreSQL
- Control de versiones: GitHub
